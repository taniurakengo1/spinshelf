Features

⌨️

Custom Shortcuts

Set your own keyboard shortcuts for rotating left and right. Default: Ctrl+Shift+Arrow keys.

🖥️

Flexible Display Order

Choose which displays participate in rotation and in what order. Exclude displays you want untouched.

✨

Smooth Animations

Foreground windows fade during cross-resolution moves. Background windows move instantly and invisibly.

🪶

Lightweight

Menu bar app with zero CPU usage when idle. No Electron, no frameworks — pure Swift.

🌏

English & Japanese

Full localization for English and Japanese. Automatically matches your system language.

🔓

Open Source

MIT licensed. Inspect the code, contribute, or fork it. No telemetry, no accounts.

Installation

1

Clone & Build

Requires macOS 13+ and Xcode Command Line Tools.

git clone https://github.com/taniurakengo1/spinshelf.git
cd spinshelf
make install
2

Grant Accessibility Permission

SpinShelf needs Accessibility access to move windows. You'll be prompted on first launch.

3

Start Rotating

Press Ctrl+Shift+→ to rotate right, Ctrl+Shift+← to rotate left. Customize in Settings.
