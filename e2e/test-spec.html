P001 アプリ起動・メニューバーアイコン表示 QAリーダー (10年経験)
Preconditions

macOS 13以降がインストールされていること。SpinShelfがビルド済みであること。アクセシビリティ権限が付与済みであること。

Test Steps
  1. ターミナルを開き、SpinShelfプロジェクトディレクトリに移動する
  2. swift run を実行してSpinShelfを起動する
  3. macOSメニューバー（画面右上）を確認する
  4. 矢印アイコン（arrow.left.arrow.right.square）がメニューバーに表示されていることを目視確認する
  5. コンソールログに「[SpinShelf] Carousel started」が出力されていることを確認する
  6. アイコンがメニューバー上で適切なサイズ（squareLength）で表示されていることを確認する
Expected Results

SpinShelfが正常に起動し、メニューバーにarrow.left.arrow.right.squareアイコンが表示される。ログに「Carousel started」が記録される。アイコンはシステムの他のメニューバーアイコンと同様のサイズで表示される。

P002 メニューからの正常終了 テスト自動化エンジニア
Preconditions

SpinShelfが起動中であること。メニューバーにアイコンが表示されていること。

Expected Results

SpinShelfプロセスがクリーンに終了する。メニューバーからアイコンが消える。クラッシュログが生成されない。NSApplication.terminate(nil) が正常に呼ばれる。
